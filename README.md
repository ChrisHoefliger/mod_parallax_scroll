This is a fork of the Joomla module Parallax Scroll written and published by https://logichunt.com/documentation/joomla-parallax-scroll.
They last updated the module back in in January 2021 and sadly discontinued it after that. As it is a simple module that comes in handy, I thought I'll try to make it fit for Joomla 5 & 6.
There it is now and runs without the campatibility plugin.
